# Одобри кредит — quick view

## Что Это

`Одобри кредит` — мини-игра в браузере про кредитного аналитика. Игрок проверяет заявки, документы и БКИ, а затем выбирает: `ОДОБРИТЬ`, `ОТКАЗ` или `ВЫЗВАТЬ СБ`.

## Как Открыть

```bash
open /Users/vaceslavboriskin/Documents/New\ project/credit_game/standalone_v3.html
```

Файл автономный: сборка и установка зависимостей не нужны.

## Как Реализована

Игра сделана одним HTML-файлом:

- HTML — структура экранов, документов и кнопок;
- CSS — ретро-визуал, адаптивная сетка, портреты, конверт заявки и анимации;
- JavaScript — генерация клиентов, логика документов, таймер, скоринг, проверка и журнал решений.

## Инструмент

Разработка и доработка выполнены в Codex. Игра работает локально в браузере без сервера.

## Версия LLM

Использовалась модель класса GPT-5. Актуальный файл игры: `standalone_v3.html`.
