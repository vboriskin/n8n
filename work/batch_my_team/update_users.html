Массовое обновление specialization + working-schedule

Открой эту страницу, будучи авторизованным на sbof-iam-prom.omega.sbrf.ru, чтобы браузер подложил куки. Если страницу открыть с file:// или с другого домена — запросы упадут на CORS/аутентификации. В этом случае используй версию-скрипт в консоли DevTools.

Правила маппинга
specialization: grade=7 → ["EXPERT"], grade=8 → ["MENTOR"], grade=0 / пусто / отсутствует / иное → [].
working-schedule: по локальному времени 09:00–18:00, обед 14:00–16:00 (пн–пт). На API уходит в UTC — вычитается timeZone часов. Пример для tz=3: workStart 06:00:00, lunchStart 11:00:00, lunchEnd 13:00:00, workEnd 15:00:00.
