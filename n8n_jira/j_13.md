## ⚙️ 1.2 Rule-based scoring (без LLM)

Добавляем ноду:

## `Code / Rule-Based Competency Scoring`

```javascript
const evidence = $json.evidence_trace || [];
const competencies = $json.competency_catalog?.competencies || [];

const results = [];

for (const comp of competencies) {
  let score = 0;

  for (const e of evidence) {
    if (comp.signals?.jira?.some(s => e.value?.includes?.(s))) {
      score += e.weight || 0.5;
    }
  }

  const level =
    score > 15 ? 5 :
    score > 10 ? 4 :
    score > 6 ? 3 :
    score > 3 ? 2 : 1;

  results.push({
    competency_id: comp.id,
    rule_based_level: level,
    score
  });
}

return [{ json: { rule_scores: results } }];
```

## ⚙️ 2.2 Grade Gap Engine

## `Code / Compute Grade Gap`

```javascript
const results = $items("Code / Normalize Competency Result").map(i => i.json);
const role = $json.employee.role_id;
const grade = $json.employee.grade_id;

const track = $json.career_tracks.roles.find(r => r.role_id === role);
const expected = track.grades.find(g => g.grade_id === grade).expected;

const gaps = [];

for (const r of results) {
  const expectedLevel = expected[r.competency_id] || 0;

  gaps.push({
    competency_id: r.competency_id,
    expected: expectedLevel,
    actual: r.estimated_level,
    gap: r.estimated_level - expectedLevel
  });
}

return [{ json: { grade_gap: gaps } }];
```
