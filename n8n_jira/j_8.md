# 🚀 Как превратить это в твою финальную систему

После импорта:

## 1. Заменяешь ноды:

* `Extract Changelog Dev` → на твой полноценный extractor
* `Aggregate Jira Summary` → на **Build Jira Summary Payload (наш финальный код)**
* добавляешь:

  * `Normalize Jira Evidence`
  * `Build Competency Payloads`
  * LLM по компетенциям

---

## 2. Добавляешь ветку компетенций

После `Aggregate Jira Summary` просто ветвишь:

```text
Split Issues
   ├─> Normalize Evidence → Competency pipeline
   └─> Jira Summary
```
