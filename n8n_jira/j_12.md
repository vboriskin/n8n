# 🧠 1. Skill Gap Intelligence (ядро всего)

## `Code / Build Skill Gap Model`

```javascript
const competencies = items.map(i => i.json);

const gaps = [];

for (const c of competencies) {
  const diff = (c.estimated_level || 0) - (c.self_level || 0);

  gaps.push({
    competency_id: c.competency_id,
    gap: diff,
    priority: c.priority,
    confidence: c.confidence,
    risk_flags: c.risk_flags || [],
    evidence_missing: c.evidence_missing || []
  });
}

return [{ json: { skill_gaps: gaps } }];
```

# 📚 3. Learning Recommendations Engine

## `Code / Match Internal Trainings`

```javascript
const gaps = $json.skill_gaps;
const trainings = $item(0).$node["Merge / Attach Jira User"].json.training_catalog || [];

const recommendations = [];

for (const g of gaps) {
  const matched = trainings.filter(t =>
    t.competency_ids?.includes(g.competency_id)
  );

  recommendations.push({
    competency_id: g.competency_id,
    internal: matched.slice(0, 3)
  });
}

return [{ json: { learning_recommendations: recommendations } }];
```
